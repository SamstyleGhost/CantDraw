SoftCodeHack 2023

Contributors:

Steps to run:
1. Clone the repository
2. Get ProdiaAI API key and VerbWire API key and insert them into credentials.js
3. Run npm install
4. Run npm run dev
5. Enter a prompt in the prompt field
6. Click on "Generate"
7. Click on "Mint" if you want to mint the resulting image
